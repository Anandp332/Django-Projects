# Django-Projects


## Create New Project
$ django-admin startproject project_name

## Create New Module
$ python manage.py startapp module_name

## For Migration
$ python manage.py makemigrations
$ python manage.py migrate

## Create SuperUser
$ python manage.py createsuperuser
* follow the instructions
